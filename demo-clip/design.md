# ToneTutor Demo Clip — Design System

Matches the live app (static/index.html score card + UI), so the clip feels like the product.

## Palette
- Background gradient: `#0D1117` → `#161B22`
- Accent / brand red: `#E5534B`
- Primary text: `#E6EDF3`
- Muted text: `#7D8590`
- Success green: `#3FB950`
- Amber (WORK ON): `#D29922`
- Card surface: `#161B22` with `#21262D` borders

## Typography
- Font: **Inter** (built-in). Headlines 700–800, body 500–600.

## Format
- Vertical **1080 × 1920** (9:16), 30s master (LAUNCH.md shot list).

## Rules / Don'ts
- No invented colors — palette above only.
- The tone-correction "money shot" (red→green flip) is the hero moment — give it the most screen time + the biggest motion.
- Placeholder screen slots are labeled `▶ SCREEN CLIP` — the user swaps in real recordings + voiceover later (WC-hook placeholder pattern). Do not treat placeholders as final.
